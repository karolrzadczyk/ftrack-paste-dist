Ftrack Paste – Privacy Policy

Last updated: May 2026

Ftrack Paste is a browser extension that allows users to paste images from the clipboard directly into ftrack Studio note editors.

Data collection

This extension does not collect, transmit, or share any personal data. Specifically:

  • No usage data or analytics are collected.
  • No clipboard content is stored or sent to any server other than the ftrack Studio instance the user is already logged into (letko.ftrackapp.com).
  • The only data stored locally is the user's enabled/disabled preference, saved in Chrome's local storage on the user's own device.

Permissions

  • storage – stores the on/off preference locally on the user's device.
  • tabs – detects navigation to the ftrack host so the extension activates only on the correct page.
  • host permission (letko.ftrackapp.com) – required to intercept paste events and upload images to the ftrack API on behalf of the logged-in user.
